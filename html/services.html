Services

Syslog

Esp-link sends event/debug info to this syslog host (hostname:port). Leave empty to disable syslog.
Stop sending syslog if free heap drops below this many bytes
Minimum severity to send
Some syslog servers rotate log if timestamp is in the past so disable to prevent this

mDNS

Esp-link can advertise its hostname and service name (both with a .local suffix) via multicast DNS.

The default service is http. For the arduino IDE use arduino

SNTP

Simple Network Time Protocol server to query. Leave empty to disable SNTP
Offset hours to apply (no daylight savings support)
